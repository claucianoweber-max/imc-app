# imc-app
## Calculadora de IMC em React

Este projeto é uma aplicação simples desenvolvida em React para calcular o **Índice de Massa Corporal (IMC)**.  
Foi criado com o objetivo de praticar **componentes, estados e eventos em React**.

---

## Funcionalidades

- Inserção de peso (kg) e altura (cm)
- Cálculo o IMC
- Exibição do resultado com 2 casas decimais
- Classificação do IMC:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade
- Exibição dos dados informados
- Alteração de cor conforme a classificação
- Botão para limpar os campos
